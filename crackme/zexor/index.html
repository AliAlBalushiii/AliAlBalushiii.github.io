We can see the strcmp() call, as we know on x64 Windows, strcmp handle parameter like that:

  • RCX reg for user input.
  • RDX reg for another located resource(which is the expected licence).

So our focus on the RDX

  • The RDX assigning:

  • So RDX is assigned to cs:LICENO, and cs indicates that LICENO is a global variable.

Let's navigate to the LICENO


As we can see here, the LICENO is a pointer pointing to the address holding the license key(dq or Quadword whic represent address in a x64 system).
